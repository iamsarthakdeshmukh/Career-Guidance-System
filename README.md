# Career Guidance System

## 📌 Project Overview
The Career Guidance System is a web-based application designed to help students choose the most suitable career path based on their interests, skills, and abilities.

The system analyzes user inputs and provides career recommendations using a Machine Learning model. It combines a modern web interface with backend APIs and an ML prediction module to deliver intelligent career suggestions.

---

## 🧠 System Architecture

The project is divided into three main components:

1. **Frontend (React)**
   - User interface
   - Collects user inputs
   - Displays career recommendations

2. **Backend (Express.js)**
   - Handles API requests
   - Communicates between frontend and ML model
   - Manages application logic

3. **Machine Learning Module (Python)**
   - Processes user responses
   - Predicts suitable career paths
   - Returns results to backend

---

## 🚀 Features

- User-friendly interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/career-guidance-system.git
```

2️⃣ Setup Frontend
cd frontend
npm install
npm start
Frontend will run on:

http://localhost:3000

3️⃣ Setup Backend

cd backend
npm install
node server.js

Backend will run on:
http://localhost:5000

4️⃣ Setup ML Module
cd ml-model
pip install -r requirements.txt
python model.py
